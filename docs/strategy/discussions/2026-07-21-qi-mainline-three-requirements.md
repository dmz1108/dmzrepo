# 主线题材三要件(Owner 定稿 2026-07-21)

## 结论

正式主线(真主线)必须同时满足三要件,宁缺毋滥:

1. **确认明星 ≥ 1**:`confirmed`(封板 + 三项取二比值闸,#188 口径)。`expected` 预期明星不再单独够格上正式榜。
2. **合格龙头 ≥ 1**:现有龙头打分体系(10日涨停次数/涨幅池排名+主因门槛综合打分)产出非空。
3. **题材属性**:候选必须是题材/概念板;**风格/指数/持仓聚合板剔除**(显式黑名单起步),它们天然产不出"题材龙头"。

`expected`-only 且有龙头的候选降级为**预备主线**(独立区展示,琥珀视觉,与 Codex #200 回看层
"未兑现候选"同构),不进正式榜、不计正式主线口径;缺龙头或缺明星证据的候选照旧只留诊断。

## 粘性规则挂靠(#123 修订)

- `confirmed` 出现过 → 明星证据侧永久保留(资金转弱不消失);正式榜资格仍需当时点龙头非空。
- `expected` 出现过 → 保留在**预备区**与回看轨迹。
- 两者都完整进入预测档案 starTransitions(预备主线的明星轨迹照常落库,命中率飞轮不断粮)。

## 边界

- 三要件门槛自 2026-07-21 起生效(`STRATEGY_MAINLINE_THREE_REQ_START_DAY`);2026-07-16~20
  的历史日维持"expected 或 confirmed"旧口径,不追溯改写(与 #123 实施日切界同一纪律)。
- 风格板黑名单同时作用于主线评选与自动扫描候选池(风格板不再消耗扫描配额);
  黑名单为显式清单,后续可升级为板型分类。
- 东财/同花顺两源独立判定不变;KPL 不参与不变;L2 明星判定本身(#188)不变。

## 拒绝的替代方案

- 仅前端隐藏风格板:扫描配额仍被浪费,拒绝。
- expected 计入正式榜但降权:Owner 明确"必须有确认明星",拒绝。
- 自动板型分类先行:样本不足易误伤,先显式黑名单。
